Unclear Model Routing is the kind of issue that usually appears after a result has already been shared. This checklist moves the review step forward so the tool output is safer to reuse.

Check the source input, the owner of the final review, the units or language used in the result, and the export destination. If any item is unclear, add a note rather than hiding the uncertainty.

A checklist page is also an SEO page when it solves a real problem. Keep it specific, operational, and tied to the working tool.
